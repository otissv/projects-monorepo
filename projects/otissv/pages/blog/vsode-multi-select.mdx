# Vscode Editing

Modifier
Windows:: ALt
Mac: Cmd

| Command                              | Windows              | Mac                   |
| ------------------------------------ | -------------------- | --------------------- |
| Add next occurrence                  | Alt + Down           | Cmd + Down            |
| Add previous occurrence              | Alt + Up             | Cmd + Up              |
| Multiple column cursor               | Ctrl + Alt + Up/Down | Cmd + Alt + Up/Down   |
| Jump to end of word                  | Ctrl + Right         | Cmd + Right           |
| Jump to beginning of word            | Ctrl + Left          | Cmd + Left            |
| Select mutiple lines                 | Ctrl +Shit + Up/Down | Cmd + Shift + Up/Down |
| Select mutiple previous word on line | Ctrl +Shit + Left    | Cmd + Shift + Left    |
| Select mutiple next word on line     | Ctrl +Shit + Right   | Cmd + Shift + Right   |
| Select whole word                    | Ctrl + D             | Cmd + D               |
| Select all instances of word         | Ctrl + Shift + L     | Cmd + Shift + L       |
| Select next occurrences              | Ctrl + K Ctrl + D    | Cmd + K Cmd + D       |
